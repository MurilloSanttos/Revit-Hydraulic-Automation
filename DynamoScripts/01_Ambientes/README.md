# Dynamo Scripts — Etapa 01: Ambientes

Esta pasta conterá scripts Dynamo (.dyn) relacionados à detecção e manipulação de ambientes.

## Planejados
- `01_ValidarRooms.dyn` — Validação visual de Rooms no modelo
- `02_CriarSpacesMassivo.dyn` — Criação em massa de Spaces (alternativa ao plugin)

## Observações
- Scripts Dynamo serão desenvolvidos conforme necessidade
- O plugin é o orquestrador principal; Dynamo é o executor paramétrico
- A comunicação entre plugin e Dynamo é feita via JSON (pasta `Data/`)
