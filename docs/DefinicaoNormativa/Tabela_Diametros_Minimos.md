## 7. Regras de Validação para o Plugin

| # | Regra | Condição | Nível |
|---|-------|---------|-------|
| V-DN-01 | DN sub-ramal AF ≥ DN mínimo do aparelho | `SE DN < tabela[aparelho].agua_fria_mm` | Crítico |
| V-DN-02 | DN ramal descarga ES ≥ DN mínimo do aparelho | `SE DN < tabela[aparelho].esgoto_mm` | Crítico |
| V-DN-03 | DN ramal do vaso ≥ 100mm | `SE DN_ramal_vaso < 100` | Crítico |
| V-DN-04 | DN subcoletor ≥ 100mm | `SE DN_subcoletor < 100` | Crítico |
| V-DN-05 | DN TQ que recebe vaso ≥ 100mm | `SE TQ_recebe_vaso E DN_TQ < 100` | Crítico |
| V-DN-06 | DN esgoto nunca diminui no escoamento | `SE DN_jusante < DN_montante` | Crítico |
| V-DN-07 | DN ventilação ≥ 2/3 DN do TQ | `SE DN_vent < 2/3 × DN_TQ` | Médio |
| V-DN-08 | DN ramal ventilação ≥ 1/2 DN ramal descarga | `SE DN_vent_ramal < 1/2 × DN_desc` | Médio |
| V-DN-09 | DN ramal ventilação ≥ 40mm | `SE DN_vent_ramal < 40` | Médio |
| V-DN-10 | DN usado é diâmetro comercial válido | `SE DN NOT IN lista_comercial` | Crítico |

## 8. Notas de Implementação

### 8.1 Algoritmo de seleção de DN mínimo

```
FUNÇÃO selecionar_dn_minimo(aparelho, sistema):
  SE sistema == "agua_fria":
    RETORNAR tabela[aparelho].agua_fria_mm
  SE sistema == "esgoto":
    RETORNAR tabela[aparelho].esgoto_mm
  SE sistema == "ventilacao":
    SE aparelho é TQ:
      RETORNAR tabela_vent_tq[DN_TQ].dn_vent_min_mm
    SENÃO:
      RETORNAR MAX(tabela[aparelho].ventilacao_mm, 40)
```

### 8.2 Algoritmo de validação de DN aplicado

```
FUNÇÃO validar_dn(trecho):
  dn_aplicado = trecho.diametro
  dn_minimo = selecionar_dn_minimo(trecho.aparelho, trecho.sistema)
  
  SE dn_aplicado < dn_minimo:
    ERRO_CRITICO("DN {dn_aplicado} < mínimo {dn_minimo} para {aparelho}")
  
  SE trecho.sistema == "esgoto":
    SE trecho.DN_jusante < trecho.DN_montante:
      ERRO_CRITICO("DN diminui no escoamento")
  
  SE dn_aplicado NOT IN diametros_comerciais[trecho.material]:
    ERRO_CRITICO("DN {dn_aplicado} não é diâmetro comercial")
  
  RETORNAR OK
```

### 8.3 Quando o plugin deve escolher automaticamente vs. perguntar

| Situação | Ação do plugin |
|----------|---------------|
| Vaso sanitário detectado no modelo | Perguntar: "Caixa acoplada ou válvula de descarga?" (padrão: caixa) |
| Pia de cozinha detectada | Perguntar: "Com ou sem triturador?" (padrão: sem) |
| DN calculado > DN mínimo | Usar DN calculado (sem perguntar) |
| DN calculado < DN mínimo | Usar DN mínimo automaticamente + log Info |
| DN não é comercial | Arredondar para o DN comercial imediatamente acima |
